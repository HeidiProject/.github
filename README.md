# Heidi Project

HEIDI is a user-friendly web interface designed for sample management and is integral for the SLS 2.0 Project. It offers user-friendly interactions and programmatic access to meet diverse experiment requirements. User-driven feature development is encouraged, and interested individuals are invited to participate in the external working group by contacting mx-software@psi.ch.

The project utilizes a microservice architecture and currently consists of the following repositories:

* backend
* frontend
* image-server
* mxdb-server
* mxdb-streaming
* nginx-reverse-proxy
